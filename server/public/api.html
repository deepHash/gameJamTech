GameJamTech API

Get Game By ID:

This is a GET route. Find game by submitted id

values: string
notes: adding '-1' at the end of the route will result in all the documents in the collection
Example:
https://e2tk8tnyw6.execute-api.eu-west-1.amazonaws.com/dev/api/games/getGamesById/12

Get Game By Name:

This is a GET route. Find game by name

values: string
Example: 
https://e2tk8tnyw6.execute-api.eu-west-1.amazonaws.com/dev/api/games/getGamesByName/ball

Update Game Name:

This is a POST route. Update an existing game name

Keys:{gameId:'id', gameName:'new name'}
Example:
https://e2tk8tnyw6.execute-api.eu-west-1.amazonaws.com/dev/api/games/updateGameName/

Insert New Game:

This is a POST route. Insert a new game document to the mongo database

Keys:{gameId: 'id', gameName: 'game name', image: 'game icon src image'}
Example: 
https://e2tk8tnyw6.execute-api.eu-west-1.amazonaws.com/dev/api/games/
